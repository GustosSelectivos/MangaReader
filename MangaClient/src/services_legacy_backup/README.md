# Backup de Services Originales (DRF)

Estos archivos son un respaldo de la capa de servicios original que apuntaba a la API en **Django REST Framework (DRF)**. 

**Ubicación original:** `MangaClient/src/services/`

Fueron respaldados debido a la migración local hacia FastAPI. En caso de requerir volver a la estructura DRF para el panel de administración o pruebas de regresión, se pueden copiar los archivos de esta carpeta de vuelta a `src/services/` sobreescribiendo los nuevos.
